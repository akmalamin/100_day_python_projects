## Secret Auction Program

A silent auction program where bidders place bids without seeing each other's bids. Uses simple newline-based screen clearing for universal compatibility.

### Features
- ✅ **Universal compatibility** - Works on all platforms (no OS dependencies)
- ✅ **Secret bidding** - Screen clears between bidders using `print("\n" * 50)`
- ✅ **Input validation** - Handles invalid amounts, empty names, and bad responses
- ✅ **Professional UI** - Formatted output with emojis and clear sections
- ✅ **Flexible responses** - Accepts `y/n/yes/no` for continuation
- ✅ **Error resistant** - Never crashes on bad user input

### How to Run
```bash
python secret_auction.py
